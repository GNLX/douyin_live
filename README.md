# douyin_live


抖音直播间数据采集  python源码、web和app都有
  
抖音直播、弹幕数据、零遗漏、高速接收直播间内容，包括来访、评论、关注、粉丝团、送礼、社评等等

---

逻辑：

使用python采集抖音直播间数据。

通过还原websocket协议来进行与服务端通讯，

通过protobuf协议来解析返回的数据，并保持于服务端的心跳连接。
